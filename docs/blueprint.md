# **App Name**: HolaMundo

## Core Features:

- Display Message: Display the text 'HOLA MUNDO' on the page.

## Style Guidelines:

- Background color: A light, desaturated blue (#D1E8FF) provides a calm backdrop.
- Primary color: A deeper blue (#4882B8) to make the text standout against the background.
- Accent color: A lighter blue (#7EB5E5) that complements the primary, creating a smooth aesthetic.
- Font: 'Inter', a versatile sans-serif font suitable for a simple message.
- The text should be large and centered on the page for optimal visibility.
- Minimalist layout with the text centered.
